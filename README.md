# SignLanguageDetection
A CPU-optimized real-time American Sign Language (ASL) alphabet recognition system using CNNs and MobileNetV2 transfer learning, featuring data augmentation, model optimization, and live webcam inference.
The project explores both **custom CNN architectures** and **transfer learning with MobileNetV2**, making it suitable for research, experimentation, and accessibility-focused applications.
